# FinanSmart: Seu Controle Financeiro Pessoal

![FinanSmart Screenshot](placeholder_screenshot.png)

## Descrição do Projeto

O FinanSmart é uma aplicação web de controle financeiro pessoal, desenvolvida com **HTML, CSS e JavaScript puros**, projetada para ser intuitiva, moderna e responsiva. Inspirado em interfaces de sistemas financeiros como Nubank e Inter, oferece uma experiência de usuário premium para gerenciar receitas e despesas de forma eficiente. Todos os dados são armazenados localmente no navegador do usuário (LocalStorage), garantindo privacidade e acesso offline. O projeto é totalmente compatível com hospedagem no GitHub Pages, sem a necessidade de um servidor backend.

## Funcionalidades

### Dashboard

*   **Saldo Atual:** Visualização clara do saldo disponível.
*   **Total de Receitas:** Soma de todas as entradas financeiras.
*   **Total de Despesas:** Soma de todos os gastos.
*   **Economia do Mês:** Cálculo percentual da economia baseada nas receitas e despesas.
*   **Cards Interativos:** Efeitos `hover` e sombras para uma experiência visual agradável.
*   **Layout Responsivo:** Adaptação perfeita a qualquer tamanho de tela.

### Controle de Transações

*   **Adicionar/Editar/Excluir:** Gerenciamento completo de receitas e despesas.
*   **Pesquisa em Tempo Real:** Encontre transações rapidamente por descrição ou categoria.
*   **Filtros Avançados:** Filtre por categoria e mês para uma análise detalhada.
*   **Ordenação:** Organize transações por data ou valor (crescente/decrescente).
*   **Detalhes da Transação:** Cada movimentação inclui descrição, valor, categoria, tipo (receita/despesa) e data.

### Categorias Padrão

*   Alimentação
*   Transporte
*   Moradia
*   Saúde
*   Educação
*   Lazer
*   Compras
*   Salário
*   Investimentos
*   Outros

### Gráficos Interativos (Chart.js)

*   **Gastos por Categoria:** Gráfico de pizza para visualizar a distribuição das despesas.
*   **Fluxo Mensal:** Gráfico de barras comparando receitas e despesas ao longo dos meses.

### Armazenamento de Dados

*   **LocalStorage:** Todos os dados são salvos no navegador, persistindo entre sessões.

### Funcionalidades Extras

*   **Modo Escuro/Claro:** Alternância de tema para conforto visual.
*   **Exportar/Importar JSON:** Faça backup e restaure seus dados financeiros.
*   **Máscara de Moeda:** Valores formatados automaticamente em Real (R$).
*   **Animações Suaves:** Transições e efeitos visuais para uma interface fluida.
*   **Mensagens de Confirmação:** Alertas ao excluir registros para evitar perdas acidentais.
*   **Notificações Elegantes:** Feedback visual para ações realizadas.

### Design

*   **Interface Moderna:** Estilo clean e profissional.
*   **Cantos Arredondados:** Elementos visuais suaves.
*   **Ícones Font Awesome:** Utilização de uma vasta biblioteca de ícones.
*   **Fonte Google Fonts (Poppins):** Tipografia moderna e legível.
*   **Gradientes Discretos:** Detalhes visuais sutis.
*   **Cards com Efeito Glassmorphism:** Elementos com transparência e desfoque.
*   **Animações de Entrada:** Elementos surgindo suavemente na tela.
*   **Navegação Adaptativa:** Menu lateral para desktops e menu inferior para dispositivos móveis.
*   **Excelente Experiência Móvel:** Otimizado para uso em smartphones e tablets.

## Tecnologias Utilizadas

*   **HTML5:** Estrutura semântica do conteúdo.
*   **CSS3:** Estilização e responsividade, incluindo variáveis CSS e `backdrop-filter` para Glassmorphism.
*   **JavaScript (ES6+):** Lógica de negócios, manipulação do DOM, gerenciamento de estado e interatividade.
*   **Chart.js:** Biblioteca para criação de gráficos dinâmicos.
*   **Font Awesome:** Biblioteca de ícones.
*   **Google Fonts:** Fonte Poppins para tipografia.
